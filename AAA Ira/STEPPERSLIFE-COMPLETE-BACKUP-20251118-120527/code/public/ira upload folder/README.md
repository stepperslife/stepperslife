# Ira Upload Folder

This folder is for storing magazine-related content and assets.

## Directory Structure

- **/images** - Magazine article images, featured photos, banners
- **/articles** - Article content, drafts, and related files
- **/assets** - Other magazine assets (logos, icons, etc.)

## Usage

Upload files to the appropriate subdirectory:
- Magazine images go in `/images`
- Article content/drafts go in `/articles`
- Other assets go in `/assets`

## Access

Files in this folder are publicly accessible at:
`https://stepperslife.com/ira upload folder/[subdirectory]/[filename]`

Example: `https://stepperslife.com/ira upload folder/images/feature-photo.jpg`
