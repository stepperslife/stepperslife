## Design Specifications

### Color Scheme:
- **Background**: #1a1a2e (Dark Navy)
- **Text**: #ffffff (White)
- **Accent**: #6c63ff (Purple)
- **Hover**: #8b84ff (Light Purple)
- **Active**: #5a52d5 (Dark Purple)
- **Borders**: #2a2a3e (Lighter Navy)

### Typography:
- **Font Family**: 'Inter', 'Segoe UI', sans-serif
- **Menu Items**: 15px, Medium (500)
- **Submenu Items**: 14px, Regular (400)
- **Section Headers**: 12px, SemiBold (600), Uppercase

### Spacing:
- **Menu Item Height**: 48px
- **Submenu Item Height**: 40px
- **Icon Size**: 20px
- **Padding**: 16px horizontal, 12px vertical
- **Sidebar Width**: 280px (Desktop), 100% (Mobile)

### Icons:
Use consistent icon library across all dashboards:
- Material Icons
- Font Awesome
- Lucide Icons

### Interaction States:
- **Default**: Semi-transparent background
- **Hover**: Light purple background (#8b84ff20)
- **Active/Selected**: Purple background (#6c63ff), Bold text
- **Disabled**: 50% opacity

## JavaScript for Interactive Submenu

```javascript
// Toggle Submenu
document.querySelectorAll('.menu-item.has-submenu').forEach(item => {
  item.addEventListener('click', (e) => {
    e.preventDefault();
    const parent = item.closest('.menu-item-group');
    
    // Close other open submenus
    document.querySelectorAll('.menu-item-group').forEach(group => {
      if (group !== parent) {
        group.classList.remove('open');
      }
    });
    
    // Toggle current submenu
    parent.classList.toggle('open');
  });
});

// Mobile Menu Toggle
const menuToggle = document.querySelector('.menu-toggle');
const sidebar = document.querySelector('.sidebar');
const sidebarOverlay = document.querySelector('.sidebar-overlay');

menuToggle?.addEventListener('click', () => {
  sidebar.classList.toggle('open');
  sidebarOverlay.classList.toggle('show');
});

sidebarOverlay?.addEventListener('click', () => {
  sidebar.classList.remove('open');
  sidebarOverlay.classList.remove('show');
});

// Set Active Menu Item
const currentPath = window.location.pathname;
document.querySelectorAll('.menu-item, .submenu-item').forEach(item => {
  if (item.getAttribute('href') === currentPath) {
    item.classList.add('active');
    
    // If submenu item, open parent
    const parentGroup = item.closest('.menu-item-group');
    if (parentGroup) {
      parentGroup.classList.add('open');
    }
  }
});
```

## Implementation Notes

### Priority Order:
1. **USER** - Most traffic, customer-facing (implement first)
2. **ORGANIZER** - Core revenue generator
3. **TEAM MEMBER & ASSOCIATE** - Distribution network
4. **STAFF** - Operational necessity
5. **ADMIN** - Platform management (can be basic initially)

### Key Features for Each:
- **Consistent design** across all dashboards
- **Role-based access control** (RBAC)
- **Responsive** mobile-first design
- **Real-time updates** for notifications
- **Quick actions** on dashboard
- **Search functionality** where applicable
- **Keyboard shortcuts** for power users

### Tech Stack Recommendations:
- **Framework**: React/Vue/Angular
- **Icons**: Lucide Icons or Font Awesome
- **Routing**: React Router or Vue Router
- **State**: Redux/Vuex or Context API
- **Styling**: Tailwind CSS or Styled Components
